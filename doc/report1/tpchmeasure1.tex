\section{概要}
\begin{itemize}
 \item iodrive 8luでraid0を構築(chunk size = 64KB)
 \item raid上にpostgresqlのtable spaceを作りこの上にDBを載せる
 \item 各tableのprimary key上にB-tree indexを構築
 \item scale factor = 100
 \item Query1-10について実行時間を測定、また1秒おきにiostatで監視
\end{itemize}
